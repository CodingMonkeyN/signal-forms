Key Takeaways

  • hidden() - conditionally hide fields based on other values
  • disabled() - conditionally disable fields
  • readonly() - make fields read-only
  • debounce() - delay model updates
  • Hidden fields don't affect parent validity
  • Disabled fields don't count toward dirty state
  • All conditions are reactive - no manual subscriptions needed

Conditional Functions 

// Hidden when not business
hidden(order.companyName, ({{ '{' }} valueOf {{ '}' }}) =>
  valueOf(order.customerType) !== 'business'
);

// Disabled for business customers
disabled(order.discountCode, ({{ '{' }} valueOf {{ '}' }}) =>
  valueOf(order.customerType) === 'business'
);

// Delay model updates by 500ms
debounce(order.notes, 500);
